# Resources

* its an asset, a physical object like buildings which can be owned (counted).
* related topics: production, capacity, trade, capacity, ...
* mainly needed for construction (and maintenance)
* they might require storage (buildings) or not (e.g. Gold can be stored "infinitely")
    * if over limit, then... lose gradually, or hard limit?
* sources/production of resources:
    * mainly by buildings (prod/cap)
    * by citizens (gold via taxes)
        * get via random events (happenings) depending on karma (and amount on wealth)
    * trading
    * income influenced by modifiers (tech/features/achievements)
* consumption of resources:
    * building
    * citizens (food)
    * trading
* resource types can be used in different "areas":
    - building
    - military (build units/weapons)
    - luxury (people happyness)
    - some resources used in several areas, e.g. iron for building big buildings and forging weapons
