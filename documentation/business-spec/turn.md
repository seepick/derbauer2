# End Turn

* the game is turn based, thus all calculations happen once hit next turn
* order relevance... TBD!
* each turn is a day (week?) to introduce concept of season; display how many years played

## Turn Content

* end-turn report
    * show which resources were produced, consumed, net gain/loss
    * only endresult/high-level; not too detailed (where it comes from, etc.)
* achievements
    * based on: history (=experience), resources/buildings/armies
* happening; see [happening](happening.md) for details
    * good/bad (depends on karma)
    * info-only/choice
    * each X-days special (1 month/year)
* notifications
    * e.g. gained new title, unlocked new feature
* being attacked; see [Defend section](military.md#defend) for details
