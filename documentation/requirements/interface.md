# Ideas

* using emojis (consistent, thus centralized!)
* random changing (but semantically similar) prompts for diversity/"freshness"
* fast UI/UX: enter and go; no long confirmation
* NO! arrow up-down navigation / menu-selection
* all possible key-strokes always presented
